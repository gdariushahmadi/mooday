# Mooday — Project Showcase

این پوشه شامل داکیومنت HTML حرفه‌ای برای ارائه به کارفرما / stakeholder است.

## درباره Mooday

Mooday یک **مارکت‌پلیس همتا به همتا (P2P / C2C)** برای زنان در امارات است
که می‌خواهند **لباس‌ها، کفش‌ها، کیف‌ها و اکسسوری‌هایی** که خریده‌اند یا
هدیه گرفته‌اند را **با قیمت کمتر به دیگران بفروشند** یا **اجاره بدهند**.

این یک پلتفرم برای فروشندگان حرفه‌ای، بوتیک‌ها، یا collector ها نیست —
هر کاربر هم می‌تواند آیتم‌های کمد خود را لیست کند و هم از کمد دیگران
خرید یا اجاره کند.

## فایل‌ها

- **`index.html`** — داکیومنت اصلی. یک فایل HTML مستقل با طراحی editorial، شامل:
  - معرفی کلی پروژه (C2C، Resell & Rent)
  - نمایش زنده هر قابلیت در phone frame (از طریق iframe)
  - معماری و stack فنی
  - گالری اسکرین‌شات تمام صفحات
  - نقشه راه (Roadmap) شامل فاز Rent mode
- **اسکرین‌شات‌ها** — تصاویر PNG با کیفیت بالا از هر view برای استفاده در گالری

## نحوه استفاده

### ۱) مشاهده با iframe زنده

داکیومنت برای نمایش iframe های زنده به dev server نیاز دارد:

```bash
npm run dev
# سپس باز کنید:
http://localhost:3001/showcase/index.html
```

### ۲) مشاهده به صورت standalone (بدون dev server)

اگر فقط می‌خواهید داکیومنت را به کارفرما بدهید، کافی‌ست فایل `index.html` را در مرورگر باز کنید. iframe ها در این حالت placeholder نشان می‌دهند ولی گالری تصاویر، معماری، و متن‌ها همچنان کامل نمایش داده می‌شوند.

### ۳) دیپلوی به عنوان صفحه استاتیک

```bash
# فایل index.html را به همراه پوشه showcase در یک CDN/S3/static host قرار دهید
```

## ساختار داکیومنت

| بخش | محتوا |
|------|--------|
| Hero | معرفی پروژه C2C + live preview اصلی |
| Concept | چهار ارزش اصلی (Resell، Rent، Discovery، Social) + آمار |
| Features | هفت feature اصلی با live preview |
| Bilingual | نمایش زبان عربی و RTL |
| Architecture | شش لایه معماری + stack |
| Gallery | ۹ اسکرین‌شات با کیفیت |
| Roadmap | شش فاز توسعه آینده (با Rent mode در فاز ۲) |

## نکته برای استفاده در جلسه

- قبل از جلسه `npm run dev` را اجرا کنید تا iframe ها زنده باشند
- روی هر phone frame می‌توانید کلیک کنید و داخل اپلیکیشن واقعی navigation کنید
- دکمه `عربي` در header اپلیکیشن، کل UI را به RTL تبدیل می‌کند
